# Step 5: Adding a timer

Now you will add a timer to your game, so that the player has to get to the island as quickly as possible.

Make sure you have selected the Stage.

![Stage thumbnail](assets/stage.png){ width="120" }

Add a new variable called ==time== to your Stage.

??? info "How to add a variable in Scratch"

    - Click on **Variables** in the Code tab, then click on **Make a Variable**.

    ![Variable blocks](assets/data-blocks.png)

    - Type in the name of your variable. You can choose whether you would like your variable to be available to all sprites, or to only this sprite. Press **OK**.

    ![Create variable](assets/create-variable.png)

    - Once you have created the variable, it will be displayed on the Stage, or you can untick the variable in the Scripts tab to hide it.

    ![Variable on the stage](assets/variable-show.png)

![Time variable annotated](assets/boat-variable-annotated.png){ width="360" }
